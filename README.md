# Sistem Informasi Pengelolaan Surat

Sistem ini dibuat menggunakan Framework : 

- PHP version 7.4.14
- Database MySQL
- Framework Bootstrap
- Javascript, Ajax

User yang ada di sistem ini ada 3 yaitu 

- Admin
- Kepala Sekolah
- Guru 

Fitur yang terdapat dalam sistem : 

- Pengelolaan Surat Masuk
- Disposisi Surat
- Pengelolaan Surat Keluar
- Pengelolaan User
